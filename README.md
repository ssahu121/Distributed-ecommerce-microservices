# Distributed E-Commerce Microservices

## Project Overview

This project is a distributed e-commerce backend system built using Java and Spring Boot microservices architecture.

The application is divided into multiple independent services such as:

* API Gateway
* Eureka Service Registry
* Product Service
* Order Service
* Payment Service
* Inventory Service

The project follows microservices architecture and event-driven communication.

---

## Tech Stack

* Java 17
* Spring Boot 3
* Spring Cloud
* Eureka Server
* API Gateway
* PostgreSQL
* MongoDB
* Apache Kafka
* Docker
* Kubernetes
* Git & GitHub

## Features

* Service Discovery using Eureka
* API Gateway Routing
* Product Management
* Order Management
* Distributed Architecture
* Event-Driven Communication
* Scalable Microservices

## Project Setup

### Clone Repository

```bash

```

### Run Eureka Server

```bash
cd eureka-server
mvn spring-boot:run
```

### Run API Gateway

```bash
cd api-gateway
mvn spring-boot:run
```

### Run Product Service

```bash
cd product-service
